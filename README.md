# Lampy Paperwork

Lampy Paperwork is the standalone paperwork app that grew out of the original RCBO label generator.

It is a browser-based, dependency-free HTML/CSS/JavaScript app for live entertainment paperwork, including:

- RCBO distro labels
- Rear labels
- Table-based data entry
- Fixture patch sheets
- Project info and power calculations

## Files

- `index.html` - current working app entry point.
- `json/` - editable reference data used by the app.
- `published_versions/` - release-ready copies by version.
- `CHANGELOG.md` - version notes for this project.

## Publishing

For GitHub Pages, publish `index.html` together with the `json/` folder.
